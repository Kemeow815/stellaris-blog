因为没挺过今年六月的又一次互联网“大扫除”，并且出了更好的模拟器管理工具「51Switch管家」，所以即日起，NS 模拟器助手将停止服务。
在停止服务后，除 GitHub / Cloudflare 下载源外，其它下载源均会关停。
如果您想继续使用本软件，请在设置中关闭自动更新，并且把下载源更改为 GitHub / Cloudflare。
如果您想把助手安装的模拟器内的游戏存档、游戏更新等迁移到其它模拟器，请在模拟器主菜单点击「文件」-「打开 Yuzu (或 Ryujinx) 文件夹」，之后将其中的内容移到其它模拟器内的对应目录。


目前的 NS 模拟器助手用户可以考虑转到 51Switch 管家： https://51switch.cn/
